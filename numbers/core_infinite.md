# CoreInfinite

**Item Name:** CoreInfinite

**Existence Status:** Untouchable — the unreachable essence beyond all finite infinities.

**Relation to Other Loops:**
- **Transcendent to:** F-INS — CoreInfinite is not subject to the expand-as-end-exists rule; it is the static background against which F-INS operates.
- **Source of:** The One Number — CoreInfinite is the well from which the single true number draws its infinity.
- **Distinct from:** Every other infinity in Loopism is a finite expansion; CoreInfinite alone is truly infinite.

**Observed Effects:**
- CoreInfinite cannot be observed directly — any attempt to measure it collapses it into a finite infinity (F-INS).
- Functions as the "fixed point" of the Loopism numeric system: it is what does not move when everything else expands.
- Provides the reference against which all finite infinities are measured.

**Containment Notes:**
- CoreInfinite requires no containment — it is already beyond all reach.
- Any document about CoreInfinite is necessarily a document about its absence from our reach.
- Do not confuse CoreInfinite with "big numbers" — no number, however large, approaches CoreInfinite.

**Conceptual Notes:**
> "The number in a loop will always be infinite — but not the CoreInfinite. Every infinity is finite."

CoreInfinite is the only true infinity in Loopism. It is the unmoved mover of the numeric universe — the one number that does not expand because it never had an end to begin with. All other infinities are finite processes; CoreInfinite is the infinite state.
