# The Start

**Item Name:** The Start

**Existence Status:** Does not exist — there is no start without an end.

**Relation to Other Loops:**
- **Linked to:** Core Loop — the Start would be the first Core Loop, but there is no first.
- **Connected to:** F-INS — the Finite-Infinite Numeric Sequence demonstrates that every beginning is a repetition.
- **Opposite to:** Non-Existent Loop — both are absences, but the Start is an absence of origin, while Non-Existent Loop is an absence of content.

**Observed Effects:**
- Looking backward from any point in the timeline, there is infinite time between the "start" and the "true end."
- Because time cannot be calculated if the end is not given, the Start is an unreachable horizon.
- Every attempt to define a beginning immediately creates a prior beginning, ad infinitum.

**Containment Notes:**
- The Start cannot be located because locating it would require an end to bound it.
- All timelines in Loopism are middle-out: there is no first frame, only an infinite regress of starts.
- To document The Start is to document its absence.

**Conceptual Notes:**
> "The Start — which doesn't exist — because there is no start without an end. Because if something starts, it will eventually repeat again. So when you look in the middle of the timeline, there will be infinite time between the start and the true end because time can't be calculated if the end isn't given."

The Start is the loop that cannot be closed because it was never opened. It is the inevitable consequence of a system where every end is a beginning and every beginning is a repetition. In Loopism, the universe has no birthday — it has always been in the middle.
