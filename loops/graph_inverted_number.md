# Graph-Inverted Number

**Item Name:** Graph-Inverted Number

**Existence Status:** Dimensional paradox — a number that exists as its own inverse mapping.

**Observed Effects:**
- When plotted, the Graph-Inverted Number traces a curve that is its own reflection across all axes simultaneously.
- It cannot be localized to a single value — it is a set of all points that are their own inverses.
- Interacting with it causes nearby numbers to flip sign or dimension.

**Containment Notes:**
- Do not attempt to calculate the Graph-Inverted Number directly; it will return the same result for every input.
- Store as a relation rather than a value.
- If a loop's numeric component becomes graph-inverted, the loop enters a self-similar fractal state that cannot be resolved.

**Conceptual Notes:**
The Graph-Inverted Number is the mathematical shadow in Loopism — a number that, when you try to pin it down, turns out to be its own opposite. It is the numeric analog of the Core Loop's disappear/reappear dynamic.
