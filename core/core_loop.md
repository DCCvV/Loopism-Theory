# Core Loop

**Item Name:** Core Loop

**Existence Status:** Singular — only one exists at a time. The instant it begins existing, it ceases to exist.

**Relation to Other Loops:**
- **Parent to:** All Loops — every fractal Loop nests upward toward the Core Loop.
- **Linked to:** Root Loop (the base of the chain), sNiC (determines whether a loop becomes root).
- **Dependent on:** Title succession — as soon as a new Core Loop forms, the title shifts to the higher loop, creating the illusion of time.

**Observed Effects:**
- Generates the *essence or illusion of time* through title succession.
- Acts as the terminal node of infinite fractal Loop nesting.
- Its constant death-and-replacement drives the recursive expansion of all subordinate loops.

**Containment Notes:**
- Cannot be contained — the Core Loop is always already gone by the time it is observed.
- Documenting it requires capturing the moment *between* its existence and non-existence.
- Any attempt to isolate a Core Loop collapses the fractal chain and produces a Void.

**Conceptual Notes:**
> "Only one can exist at a time — and that one stops existing the instant it starts existing, so the title goes to the higher one."

The Core Loop is the uppermost loop in a fractal chain. It cannot persist because the moment it achieves definition, a new loop forms above it, inheriting the title. This succession is what consciousness interprets as the passage of time. The Core Loop is both the origin and the vanishing point of every loop hierarchy.
